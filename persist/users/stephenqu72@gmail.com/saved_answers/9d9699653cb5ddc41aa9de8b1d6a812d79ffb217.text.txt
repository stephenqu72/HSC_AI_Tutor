### Part a(i)
Given $z = a - 2bi$, the conjugate is $\bar{z} = a + 2bi$. We seek to find $\frac{1}{\bar{z}}$ in the form $x + iy$:
$$\frac{1}{\bar{z}} = \frac{1}{a + 2bi}$$
Multiply the numerator and denominator by the conjugate of the denominator, $(a - 2bi)$:
$$\frac{1}{a + 2bi} \cdot \frac{a - 2bi}{a - 2bi} = \frac{a - 2bi}{a^2 - (2bi)^2} = \frac{a - 2bi}{a^2 + 4b^2}$$
Separating into real and imaginary parts:
$$\frac{1}{\bar{z}} = \frac{a}{a^2 + 4b^2} - \frac{2b}{a^2 + 4b^2}i$$
**Final Answer:** $\frac{a}{a^2 + 4b^2} - \frac{2b}{a^2 + 4b^2}i$

$$z^2 = a^2 - 4abi - 4b^2$$
Grouping real and imaginary parts:
$$z^2 = (a^2 - 4b^2) - 4abi$$
**Final Answer:** $(a^2 - 4b^2) - 4abi$

### Part b(i)
Given Euler's formula $e^{i\theta} = \cos \theta + i \sin \theta$. To find $e^{-i\theta}$, substitute $-\theta$ for $\theta$:
$$e^{-i\theta} = \cos(-\theta) + i \sin(-\theta)$$
Since $\cos$ is an even function ($\cos(-\theta) = \cos \theta$) and $\sin$ is an odd function ($\sin(-\theta) = -\sin \theta$):
**Final Answer:** $e^{-i\theta} = \cos \theta - i \sin \theta$

### Part b(ii)
Starting with the expressions from (i):
$e^{i\theta} = \cos \theta + i \sin \theta$
$e^{-i\theta} = \cos \theta - i \sin \theta$
Subtract the second equation from the first:
$$e^{i\theta} - e^{-i\theta} = (\cos \theta + i \sin \theta) - (\cos \theta - i \sin \theta)$$
$$e^{i\theta} - e^{-i\theta} = 2i \sin \theta$$
Dividing by $2i$:
**Final Answer:** $\sin \theta = \frac{1}{2i}(e^{i\theta} - e^{-i\theta})$

$$\sin^3 \theta = \frac{1}{-8i} \left( e^{i3\theta} - 3e^{i\theta} + 3e^{-i\theta} - e^{-i3\theta} \right)$$
Rearrange terms:
$$\sin^3 \theta = \frac{1}{-8i} \left( (e^{i3\theta} - e^{-i3\theta}) - 3(e^{i\theta} - e^{-i\theta}) \right)$$
From part (ii), we know $\sin(n\theta) = \frac{1}{2i}(e^{in\theta} - e^{-in\theta})$, so $(e^{in\theta} - e^{-in\theta}) = 2i \sin(n\theta)$:

**Proof by contradiction:**
Assume that for an irrational number $n$, $\sqrt{n}$ is rational.
If $\sqrt{n}$ is rational, then $\sqrt{n} = \frac{p}{q}$ for some integers $p, q$ ($q \neq 0$) in simplest form.
Squaring both sides gives $n = \frac{p^2}{q^2}$, which implies $n = \frac{p^2}{q^2}$.
This implies $n$ is a ratio of two integers, which by definition means $n$ is a rational number.
This contradicts the given premise that $n$ is an irrational number.
Therefore, the assumption must be false, and $\sqrt{n}$ must be irrational.
**Final Answer:** Proved by contradiction.
