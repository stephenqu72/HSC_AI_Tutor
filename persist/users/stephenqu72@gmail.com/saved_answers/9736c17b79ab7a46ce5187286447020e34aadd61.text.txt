To solve this vector geometry problem, we represent the parallelogram $OABC$ using the position vectors $\vec{OA} = \mathbf{a}$ and $\vec{OC} = \mathbf{c}$. Since $OABC$ is a parallelogram, we have $\vec{OB} = \vec{OA} + \vec{OC} = \mathbf{a} + \mathbf{c}$. Furthermore, $\vec{AB} = \vec{OC} = \mathbf{c}$.

### Part i

We calculate the requested vectors systematically:

1.  **Vector $\vec{AC}$**:
    Using the triangle law of addition, $\vec{AC} = \vec{AO} + \vec{OC}$.
    Since $\vec{AO} = -\vec{OA} = -\mathbf{a}$, we have:
    $$\vec{AC} = \mathbf{c} - \mathbf{a}$$

2.  **Vector $\vec{AN}$**:
    We are given $AN:NC = 1:2$, which means $N$ divides $AC$ in the ratio $1:2$. Thus, $\vec{AN} = \frac{1}{1+2}\vec{AC} = \frac{1}{3}\vec{AC}$.
    Substituting $\vec{AC} = \mathbf{c} - \mathbf{a}$:
    $$\vec{AN} = \frac{1}{3}(\mathbf{c} - \mathbf{a})$$

3.  **Vector $\vec{ON}$**:
    Using the triangle law, $\vec{ON} = \vec{OA} + \vec{AN}$.
    Substituting $\vec{OA} = \mathbf{a}$ and $\vec{AN} = \frac{1}{3}\mathbf{c} - \frac{1}{3}\mathbf{a}$:
    $$\vec{ON} = \mathbf{a} + \frac{1}{3}\mathbf{c} - \frac{1}{3}\mathbf{a} = \frac{2}{3}\mathbf{a} + \frac{1}{3}\mathbf{c} = \frac{1}{3}(2\mathbf{a} + \mathbf{c})$$

4.  **Vector $\vec{NM}$**:
    Since $M$ is the midpoint of $AB$, $\vec{AM} = \frac{1}{2}\vec{AB} = \frac{1}{2}\mathbf{c}$.
    Using vector subtraction, $\vec{NM} = \vec{AM} - \vec{AN}$.
    Substituting $\vec{AM} = \frac{1}{2}\mathbf{c}$ and $\vec{AN} = \frac{1}{3}\mathbf{c} - \frac{1}{3}\mathbf{a}$:
    $$\vec{NM} = \frac{1}{2}\mathbf{c} - (\frac{1}{3}\mathbf{c} - \frac{1}{3}\mathbf{a}) = \frac{1}{3}\mathbf{a} + (\frac{1}{2} - \frac{1}{3})\mathbf{c} = \frac{1}{3}\mathbf{a} + \frac{1}{6}\mathbf{c}$$
    Factoring out $\frac{1}{6}$:
    $$\vec{NM} = \frac{1}{6}(2\mathbf{a} + \mathbf{c})$$

**Final Answer i:** $\vec{AC} = \mathbf{c} - \mathbf{a}$, $\vec{AN} = \frac{1}{3}(\mathbf{c} - \mathbf{a})$, $\vec{ON} = \frac{1}{3}(2\mathbf{a} + \mathbf{c})$, $\vec{NM} = \frac{1}{6}(2\mathbf{a} + \mathbf{c})$.

### Part ii

To prove that points $O, N,$ and $M$ are collinear, we must show that one vector joining these points is a scalar multiple of the other (e.g., $\vec{OM} = k\vec{ON}$).

First, we find $\vec{OM}$:
$\vec{OM} = \vec{OA} + \vec{AM} = \mathbf{a} + \frac{1}{2}\mathbf{c} = \frac{1}{2}(2\mathbf{a} + \mathbf{c})$.

Now, compare $\vec{OM}$ with $\vec{ON}$ from Part (i):
$\vec{ON} = \frac{1}{3}(2\mathbf{a} + \mathbf{c}) \implies (2\mathbf{a} + \mathbf{c}) = 3\vec{ON}$.

Substituting this into the expression for $\vec{OM}$:
$\vec{OM} = \frac{1}{2}(3\vec{ON}) = \frac{3}{2}\vec{ON}$.

Since $\vec{OM}$ is a scalar multiple of $\vec{ON}$ and both share the common point $O$, the points $O, N,$ and $M$ must lie on the same straight line.

**Final Answer ii:** Since $\vec{OM} = \frac{3}{2}\vec{ON}$, the vectors are parallel and share a common origin, hence $O, N,$ and $M$ are collinear.
