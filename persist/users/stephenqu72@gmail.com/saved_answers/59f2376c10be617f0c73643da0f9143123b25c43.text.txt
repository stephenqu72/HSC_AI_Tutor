To solve for the displacement function $x(t)$, we model the motion using Newton's Second Law and the kinematics of particles.

### Step 1: Establish the equation of motion
Given mass $m = 1 \text{ kg}$ and force $F = 8x^3 + 36x$, by Newton's Second Law ($F = ma$):
$$a = \frac{dv}{dt} = v \frac{dv}{dx} = 8x^3 + 36x$$
We use the expression $v \frac{dv}{dx}$ because the force is given as a function of displacement $x$.

### Step 2: Solve for velocity $v$ as a function of $x$
We separate the variables and integrate with respect to $x$:
$$\int v \, dv = \int (8x^3 + 36x) \, dx$$
$$\frac{1}{2}v^2 = 2x^4 + 18x^2 + C$$
Using the initial conditions: at $t = 0$, $x = 1$ and $v = 11$:
$$\frac{1}{2}(11)^2 = 2(1)^4 + 18(1)^2 + C$$
$$\frac{121}{2} = 2 + 18 + C \implies 60.5 = 20 + C \implies C = 40.5 = \frac{81}{2}$$
Substitute $C$ back into the equation:
$$\frac{1}{2}v^2 = 2x^4 + 18x^2 + \frac{81}{2}$$
$$v^2 = 4x^4 + 36x^2 + 81$$
Observe that the right side is a perfect square $(2x^2 + 9)^2$:
$$v^2 = (2x^2 + 9)^2$$
Since the body is moving to the right ($v > 0$), we take the positive root:
$$\frac{dx}{dt} = 2x^2 + 9$$

### Step 3: Solve for $x(t)$
Separate the variables again to find $x(t)$:
$$\int \frac{dx}{2x^2 + 9} = \int dt$$
Factor out a 2 from the denominator:
$$\frac{1}{2} \int \frac{dx}{x^2 + \left(\frac{3}{\sqrt{2}}\right)^2} = \int dt$$
Using the standard integral $\int \frac{du}{u^2 + a^2} = \frac{1}{a} \arctan(\frac{u}{a}) + C$:
$$\frac{1}{2} \cdot \frac{1}{\frac{3}{\sqrt{2}}} \arctan\left(\frac{x}{\frac{3}{\sqrt{2}}}\right) = t + C_2$$
$$\frac{\sqrt{2}}{6} \arctan\left(\frac{\sqrt{2}x}{3}\right) = t + C_2$$
At $t = 0, x = 1$:
$$\frac{\sqrt{2}}{6} \arctan\left(\frac{\sqrt{2}}{3}\right) = C_2$$
So, $\frac{\sqrt{2}}{6} \arctan\left(\frac{\sqrt{2}x}{3}\right) = t + \frac{\sqrt{2}}{6} \arctan\left(\frac{\sqrt{2}}{3}\right)$.
Isolating $x$:
$$\arctan\left(\frac{\sqrt{2}x}{3}\right) = \frac{6t}{\sqrt{2}} + \arctan\left(\frac{\sqrt{2}}{3}\right) = 3\sqrt{2}t + \arctan\left(\frac{\sqrt{2}}{3}\right)$$
$$\frac{\sqrt{2}x}{3} = \tan\left(3\sqrt{2}t + \arctan\left(\frac{\sqrt{2}}{3}\right)\right)$$
$$x(t) = \frac{3}{\sqrt{2}} \tan\left(3\sqrt{2}t + \arctan\left(\frac{\sqrt{2}}{3}\right)\right)$$

### Final Answer:
$$x(t) = \frac{3\sqrt{2}}{2} \tan\left(3\sqrt{2}t + \arctan\left(\frac{\sqrt{2}}{3}\right)\right)$$
