To determine which scenario is NOT a counterexample to Kimie's conclusion, we must identify the case where the actual average speed equals the arithmetic mean of the speeds:
$$\text{Average Speed}_{\text{total}} = \frac{\text{Total Distance}}{\text{Total Time}} = \frac{U+V}{2}$$

### Analysis of the Physics
The formula $\frac{U+V}{2}$ only holds true if the car spends an **equal amount of time** in each part of the journey. Let $t_1$ and $t_2$ be the times for the two parts, and $d_1$ and $d_2$ be the distances.
$$\text{Average Speed} = \frac{d_1 + d_2}{t_1 + t_2} = \frac{V t_1 + U t_2}{t_1 + t_2}$$
Setting this equal to $\frac{V+U}{2}$:
$$\frac{V t_1 + U t_2}{t_1 + t_2} = \frac{V+U}{2} \implies 2V t_1 + 2U t_2 = V t_1 + V t_2 + U t_1 + U t_2$$
$$V t_1 - V t_2 - U t_1 + U t_2 = 0 \implies V(t_1 - t_2) - U(t_1 - t_2) = 0 \implies (V-U)(t_1-t_2) = 0$$
Thus, the conclusion is valid only if $V=U$ or $t_1 = t_2$.

### Evaluating the Options

**Option A:** Distance $d_1 = 100\text{ km}, V = 20\text{ km/h} \implies t_1 = 5\text{ h}$. Distance $d_2 = 100\text{ km}, U = 40\text{ km/h} \implies t_2 = 2.5\text{ h}$.
Since $t_1 \neq t_2$, this is a counterexample.

**Option B:** Distance $d_1 = 120\text{ km}, V = 60\text{ km/h} \implies t_1 = 2\text{ h}$. Distance $d_2 = 80\text{ km}, U = 40\text{ km/h} \implies t_2 = 2\text{ h}$.
Here, $t_1 = t_2 = 2\text{ hours}$. Let's check the average speed:
$$\text{Total Distance} = 120 + 80 = 200\text{ km}$$
$$\text{Total Time} = 2 + 2 = 4\text{ hours}$$
$$\text{Average Speed} = \frac{200}{4} = 50\text{ km/h}$$
$$\text{Kimie's formula} = \frac{60 + 40}{2} = 50\text{ km/h}$$
Since the results match, this is **not** a counterexample.

**Option C:** $t_1 = 1\text{ h}, V = 100\text{ km/h} \implies d_1 = 100\text{ km}$. $t_2 = 2\text{ h}, U = 50\text{ km/h} \implies d_2 = 100\text{ km}$.
$\text{Average Speed} = \frac{200}{3} \approx 66.7\text{ km/h}$.
$\text{Kimie's formula} = \frac{100+50}{2} = 75\text{ km/h}$.
Since $66.7 \neq 75$, this is a counterexample.

**Option D:** $d_1 = 60, t_1 = 2 \implies V = 30\text{ km/h}$. $d_2 = 40, t_2 = 3 \implies U = 13.33\text{ km/h}$.
$\text{Average Speed} = \frac{100}{5} = 20\text{ km/h}$.
$\text{Kimie's formula} = \frac{30+13.33}{2} = 21.66\text{ km/h}$.
Since $20 \neq 21.66$, this is a counterexample.

**Final Answer:** The correct choice is **B**.
