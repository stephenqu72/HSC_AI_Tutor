To find the magnitude of the vector $\mathbf{v} = \sin \theta \mathbf{i} + \cot \theta \mathbf{j} + \cos \theta \mathbf{k}$, we use the definition of the magnitude of a 3D vector $\mathbf{v} = a\mathbf{i} + b\mathbf{j} + c\mathbf{k}$, which is given by $|\mathbf{v}| = \sqrt{a^2 + b^2 + c^2}$.

### Step-by-Step Derivation

1. **Set up the magnitude expression:**
   The magnitude squared is:
   $$|\mathbf{v}|^2 = (\sin \theta)^2 + (\cot \theta)^2 + (\cos \theta)^2$$
   $$|\mathbf{v}|^2 = \sin^2 \theta + \cot^2 \theta + \cos^2 \theta$$

2. **Apply trigonometric identities:**
   Recall the Pythagorean identity $\sin^2 \theta + \cos^2 \theta = 1$. Substituting this into our expression:
   $$|\mathbf{v}|^2 = (\sin^2 \theta + \cos^2 \theta) + \cot^2 \theta$$
   $$|\mathbf{v}|^2 = 1 + \cot^2 \theta$$

3. **Simplify using the identity for cotangent:**
   Recall the trigonometric identity $1 + \cot^2 \theta = \csc^2 \theta$.
   $$|\mathbf{v}|^2 = \csc^2 \theta$$

4. **Calculate the square root:**
   Taking the square root of both sides:
   $$|\mathbf{v}| = \sqrt{\csc^2 \theta} = |\csc \theta|$$
   Given the domain $0 < \theta < \frac{\pi}{2}$, the angle is in the first quadrant where $\sin \theta > 0$. Since $\csc \theta = \frac{1}{\sin \theta}$, $\csc \theta$ is also positive in this domain. Therefore, $|\csc \theta| = \csc \theta$.

### Conclusion
The magnitude of the vector is $\csc \theta$. Comparing this result to the given options:
A. $1$
B. $\csc \theta$
C. $\cot \theta$
D. $\sec \theta$

The correct option is **B**.
