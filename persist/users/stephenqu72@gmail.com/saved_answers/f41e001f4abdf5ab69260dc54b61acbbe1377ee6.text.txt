### Part (a)
**To prove:** $u_n = \frac{1}{3}(2^{n+1} + (-1)^n)$ for $n \in \mathbb{Z}^+$, given $u_1 = 1$ and $u_{n+1} = 2u_n + (-1)^{n+1}$.

**Step 1: Base Case**
For $n = 1$:
LHS $= u_1 = 1$.
RHS $= \frac{1}{3}(2^{1+1} + (-1)^1) = \frac{1}{3}(4 - 1) = \frac{3}{3} = 1$.
Since LHS = RHS, the statement is true for $n=1$.

**Step 2: Inductive Step**
Assume the statement is true for some positive integer $k$. That is, assume:
$u_k = \frac{1}{3}(2^{k+1} + (-1)^k)$

We must show it is true for $n = k+1$, i.e., $u_{k+1} = \frac{1}{3}(2^{k+2} + (-1)^{k+1})$.
Using the recurrence relation $u_{k+1} = 2u_k + (-1)^{k+1}$:
$u_{k+1} = 2 \left[ \frac{1}{3}(2^{k+1} + (-1)^k) \right] + (-1)^{k+1}$
$u_{k+1} = \frac{1}{3}(2 \cdot 2^{k+1} + 2(-1)^k) + (-1)^{k+1}$
$u_{k+1} = \frac{1}{3}(2^{k+2}) + \frac{2}{3}(-1)^k + (-1)^{k+1}$

Since $(-1)^{k+1} = -(-1)^k$:
$u_{k+1} = \frac{1}{3}(2^{k+2}) + (-1)^k \left( \frac{2}{3} - 1 \right)$
$u_{k+1} = \frac{1}{3}(2^{k+2}) + (-1)^k \left( -\frac{1}{3} \right)$
$u_{k+1} = \frac{1}{3}(2^{k+2}) - \frac{1}{3}(-1)^k = \frac{1}{3}(2^{k+2} + (-1)^{k+1})$

**Conclusion:** The statement is true for $n=k+1$. By the principle of mathematical induction, the formula holds for all $n \in \mathbb{Z}^+$.

### Part (b)(i)
**To show:** $\int_0^a f(x) dx = \int_0^a f(a-x) dx$.

Let $u = a - x$. Then $du = -dx$, or $dx = -du$.
Change the limits:
When $x = 0$, $u = a$.
When $x = a$, $u = 0$.

Substituting these into the integral:
$\int_0^a f(x) dx = \int_a^0 f(a-u) (-du)$
Using the property $\int_a^b = -\int_b^a$:
$= \int_0^a f(a-u) du$
Since the variable of integration is a dummy variable, we can replace $u$ with $x$:
$= \int_0^a f(a-x) dx$
**Result:** $\int_0^a f(x) dx = \int_0^a f(a-x) dx$.

### Part (b)(ii)
**Evaluate:** $I = \int_0^{\pi/4} \frac{\sin 2x}{\sin 2x + \cos 2x} dx$.

Let $a = \pi/4$. Using the result from (i), $I = \int_0^{\pi/4} \frac{\sin(2(\pi/4 - x))}{\sin(2(\pi/4 - x)) + \cos(2(\pi/4 - x))} dx$.
$I = \int_0^{\pi/4} \frac{\sin(\pi/2 - 2x)}{\sin(\pi/2 - 2x) + \cos(\pi/2 - 2x)} dx$.
Using the co-function identities $\sin(\pi/2 - \theta) = \cos \theta$ and $\cos(\pi/2 - \theta) = \sin \theta$:
$I = \int_0^{\pi/4} \frac{\cos 2x}{\cos 2x + \sin 2x} dx$.

Add the two expressions for $I$:
$2I = \int_0^{\pi/4} \frac{\sin 2x}{\sin 2x + \cos 2x} dx + \int_0^{\pi/4} \frac{\cos 2x}{\sin 2x + \cos 2x} dx$
$2I = \int_0^{\pi/4} \frac{\sin 2x + \cos 2x}{\sin 2x + \cos 2x} dx = \int_0^{\pi/4} 1 dx$
$2I = [x]_0^{\pi/4} = \pi/4$
$I = \pi/8$.

**Final Answer:** $I = \frac{\pi}{8}$.
