### Part i
We are given $S = (\sqrt{50} + 7)^{2000} + (\sqrt{50} - 7)^{2000}$.
Using the Binomial Theorem, $(a+b)^n = \sum_{k=0}^{n} \binom{n}{k} a^k b^{n-k}$, we expand the terms:
1. $(\sqrt{50} + 7)^{2000} = \sum_{k=0}^{2000} \binom{2000}{k} (\sqrt{50})^k (7)^{2000-k}$
2. $(\sqrt{50} - 7)^{2000} = \sum_{k=0}^{2000} \binom{2000}{k} (\sqrt{50})^k (-7)^{2000-k}$

Summing these expressions:
$$S = \sum_{k=0}^{2000} \binom{2000}{k} (\sqrt{50})^k \left( 7^{2000-k} + (-1)^{2000-k} 7^{2000-k} \right)$$
If $k$ is odd, $(\sqrt{50})^k$ contains an odd power of $\sqrt{50}$, resulting in an irrational term. However, the terms $7^{2000-k} + (-1)^{2000-k} 7^{2000-k}$ will be $7^m - 7^m = 0$ for all odd $k$. Thus, all irrational terms cancel out.
If $k$ is even, let $k = 2j$. Then $(\sqrt{50})^k = (50)^j$, which is an integer. The coefficient $7^{2000-k} + (-1)^{2000-k} 7^{2000-k} = 2 \cdot 7^{2000-k}$.
Thus, $S = 2 \sum_{j=0}^{1000} \binom{2000}{2j} 50^j 7^{2000-2j}$, which is clearly an integer multiplied by 2.

**Final Answer:** $S$ is an even integer.
