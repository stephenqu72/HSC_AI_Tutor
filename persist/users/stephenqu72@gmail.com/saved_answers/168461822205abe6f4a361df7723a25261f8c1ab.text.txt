### Part (a)
We are given $z = r(\cos \theta + i \sin \theta)$. Using De Moivre's Theorem, $z^n = r^n(\cos(n\theta) + i \sin(n\theta))$. The complex conjugate is $\bar{z} = r(\cos(-\theta) + i \sin(-\theta)) = r(\cos \theta - i \sin \theta)$.

The equation is $z^5 = \bar{z}$:
$$r^5(\cos 5\theta + i \sin 5\theta) = r(\cos(-\theta) + i \sin(-\theta))$$

Equating the moduli:
$r^5 = r \implies r(r^4 - 1) = 0$. Since $r > 0$ for non-zero solutions, $r = 1$ ($r=0$ gives the trivial solution $z=0$).

Equating arguments:
$5\theta = -\theta + 2k\pi$ for $k \in \mathbb{Z}$
$6\theta = 2k\pi \implies \theta = \frac{k\pi}{3}$ for $k = 0, 1, 2, 3, 4, 5$.

The solutions for $z$ are:
- $k=0: z = \cos 0 + i \sin 0 = 1$
- $k=1: z = \cos \frac{\pi}{3} + i \sin \frac{\pi}{3} = \frac{1}{2} + i\frac{\sqrt{3}}{2}$
- $k=2: z = \cos \frac{2\pi}{3} + i \sin \frac{2\pi}{3} = -\frac{1}{2} + i\frac{\sqrt{3}}{2}$
- $k=3: z = \cos \pi + i \sin \pi = -1$
- $k=4: z = \cos \frac{4\pi}{3} + i \sin \frac{4\pi}{3} = -\frac{1}{2} - i\frac{\sqrt{3}}{2}$
- $k=5: z = \cos \frac{5\pi}{3} + i \sin \frac{5\pi}{3} = \frac{1}{2} - i\frac{\sqrt{3}}{2}$
And the trivial solution $z=0$.

**Final Answer:** $z = 0, \pm 1, \pm \frac{1}{2} \pm i\frac{\sqrt{3}}{2}$

### Part (b)(i)
We wish to find the reduction formula for $I_n = \int \csc^n x \, dx$. Rewrite the integral as:
$$I_n = \int \csc^{n-2} x \cdot \csc^2 x \, dx$$
Using integration by parts, let $u = \csc^{n-2} x$ and $dv = \csc^2 x \, dx$.
Then $du = (n-2)\csc^{n-3} x (-\csc x \cot x) \, dx = -(n-2)\csc^{n-2} x \cot x \, dx$ and $v = -\cot x$.

By the formula $\int u \, dv = uv - \int v \, du$:
$$I_n = -\csc^{n-2} x \cot x - \int (-\cot x)(-(n-2)\csc^{n-2} x \cot x) \, dx$$
$$I_n = -\csc^{n-2} x \cot x - (n-2) \int \csc^{n-2} x \cot^2 x \, dx$$
Using the identity $\cot^2 x = \csc^2 x - 1$:
$$I_n = -\csc^{n-2} x \cot x - (n-2) \int \csc^{n-2} x (\csc^2 x - 1) \, dx$$
$$I_n = -\csc^{n-2} x \cot x - (n-2) I_n + (n-2) I_{n-2}$$
Add $(n-2) I_n$ to both sides:
$$(n-1) I_n = -\csc^{n-2} x \cot x + (n-2) I_{n-2}$$
$$I_n = \frac{-\cot x \csc^{n-2} x}{n-1} + \frac{n-2}{n-1} I_{n-2}$$

### Part (b)(ii)
Using the result from (i) for $n=4$:
$$I_4 = \int \csc^4 x \, dx = \frac{-\cot x \csc^2 x}{3} + \frac{2}{3} I_2$$
We know $I_2 = \int \csc^2 x \, dx = -\cot x + C$.
Thus, $I_4 = -\frac{1}{3}\cot x \csc^2 x - \frac{2}{3} \cot x + C$.
Evaluating from $\frac{\pi}{6}$ to $\frac{5\pi}{6}$:
$\left[ -\frac{1}{3}\cot x \csc^2 x - \frac{2}{3} \cot x \right]_{\pi/6}^{5\pi/6}$
At $\frac{5\pi}{6}: \cot \frac{5\pi}{6} = -\sqrt{3}, \csc \frac{5\pi}{6} = 2$.
Value $= -\frac{1}{3}(-\sqrt{3})(4) - \frac{2}{3}(-\sqrt{3}) = \frac{4\sqrt{3}}{3} + \frac{2\sqrt{3}}{3} = 2\sqrt{3}$.
At $\frac{\pi}{6}: \cot \frac{\pi}{6} = \sqrt{3}, \csc \frac{\pi}{6} = 2$.
Value $= -\frac{1}{3}(\sqrt{3})(4) - \frac{2}{3}(\sqrt{3}) = -\frac{4\sqrt{3}}{3} - \frac{2\sqrt{3}}{3} = -2\sqrt{3}$.
Result $= 2\sqrt{3} - (-2\sqrt{3}) = 4\sqrt{3}$.

**Final Answer:** $4\sqrt{3}$
