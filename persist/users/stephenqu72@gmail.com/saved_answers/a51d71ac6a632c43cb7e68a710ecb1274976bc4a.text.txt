### Logical Analysis

To determine if the rule $P \implies Q$ holds true, we must identify which cases could potentially falsify the statement. An implication $P \implies Q$ is **only false** when $P$ is true and $Q$ is false (i.e., the presence of a tree paired with a colour that is *not* yellow).

### Selecting the Method
To verify the truth of a universal conditional statement, we are looking for evidence that could disprove it. If we find a case where the antecedent ($P$) is true but the consequent ($Q$) is false, we have identified a **counterexample**.

*   We check the **Tree** card to see if $Q$ follows.
*   We check the **Blue** (non-yellow) card to ensure it is not a tree ($P$).

If we find a card that is a tree but not yellow, we have successfully demonstrated the rule is false via a counterexample. The process of testing a logical rule by looking for instances where the rule fails is the application of searching for a **counterexample**.
