### Part a
To prove that $\sqrt[3]{2}$ is irrational, we use a proof by contradiction.

Assume $\sqrt[3]{2}$ is rational. Then there exist coprime positive integers $p$ and $q$ such that:
$$\sqrt[3]{2} = \frac{p}{q}$$
Cubing both sides:
$$2 = \frac{p^3}{q^3} \implies p^3 = 2q^3$$
This implies $p^3$ is even, so $p$ must be even (let $p = 2k$ for some integer $k$). Substituting this back:
$$(2k)^3 = 2q^3 \implies 8k^3 = 2q^3 \implies q^3 = 4k^3$$
This implies $q^3$ is even, so $q$ must be even. If both $p$ and $q$ are even, they share a common factor of 2, which contradicts our assumption that $p$ and $q$ are coprime. Thus, our initial assumption is false.
**Conclusion:** $\sqrt[3]{2}$ is irrational.

### Part b
Given $(x + i)^4$ is purely imaginary, let $z = x + i$. We expand using the Binomial Theorem:
$$(x + i)^4 = \binom{4}{0}x^4 + \binom{4}{1}x^3(i) + \binom{4}{2}x^2(i^2) + \binom{4}{3}x(i^3) + \binom{4}{4}(i^4)$$
$$= x^4 + 4ix^3 - 6x^2 - 4ix + 1$$
Group the real and imaginary parts:
$$(x^4 - 6x^2 + 1) + i(4x^3 - 4x)$$
For a number to be purely imaginary, the real part must equal zero:
$$x^4 - 6x^2 + 1 = 0$$
Let $u = x^2$. Then $u^2 - 6u + 1 = 0$. Using the quadratic formula:
$$u = \frac{6 \pm \sqrt{36 - 4}}{2} = 3 \pm \sqrt{8} = 3 \pm 2\sqrt{2}$$
Since $x^2 = 3 \pm 2\sqrt{2}$, we note that $(\sqrt{2} \pm 1)^2 = 2 \pm 2\sqrt{2} + 1 = 3 \pm 2\sqrt{2}$.
Therefore, $x = \pm(\sqrt{2} + 1)$ and $x = \pm(\sqrt{2} - 1)$.
**Final answer:** $x = \pm(\sqrt{2} + 1), \pm(\sqrt{2} - 1)$.

### Part c
We solve $z^5 = 4 + 4i$. First, convert $4 + 4i$ to polar form:
Modulus $r = \sqrt{4^2 + 4^2} = \sqrt{32} = 4\sqrt{2} = 2^{5/2}$.
Argument $\theta = \arctan(4/4) = \pi/4$.
So, $z^5 = 2^{5/2} e^{i(\pi/4 + 2k\pi)}$ for $k = 0, 1, 2, 3, 4$.
Applying De Moivre's Theorem:
$$z = (2^{5/2})^{1/5} e^{i(\pi/20 + 2k\pi/5)} = \sqrt{2} e^{i(\frac{1+8k}{20})\pi}$$
The solutions are:
$k=0: \sqrt{2}e^{i\pi/20}$
$k=1: \sqrt{2}e^{i9\pi/20}$
$k=2: \sqrt{2}e^{i17\pi/20}$
$k=3: \sqrt{2}e^{i25\pi/20} = \sqrt{2}e^{i5\pi/4}$
$k=4: \sqrt{2}e^{i33\pi/20}$
**Final answer:** $z = \sqrt{2} \exp\left(i\frac{(8k+1)\pi}{20}\right)$ for $k=0, 1, 2, 3, 4$.

### Part d
i) The locus $|z - (\sqrt{3} + i)| = 1$ is a circle centered at $C = (\sqrt{3}, 1)$ with radius $r = 1$.
ii) The center $C$ is at $1 + i\sqrt{3}$? No, wait: $C = \sqrt{3} + i$.
Distance from origin to $C$ is $|C| = \sqrt{(\sqrt{3})^2 + 1^2} = \sqrt{4} = 2$.
Max $|z| = |C| + r = 2 + 1 = 3$.
For $\text{Arg}(z)$, the tangent lines from the origin to the circle form an angle $\alpha$ with the line $OC$.
$\sin \alpha = r / |C| = 1/2 \implies \alpha = \pi/6$.
$\text{Arg}(C) = \arctan(1/\sqrt{3}) = \pi/6$.
Max $\text{Arg}(z) = \text{Arg}(C) + \alpha = \pi/6 + \pi/6 = \pi/3$.
**Final answer:** Max $|z| = 3$, Max $\text{Arg}(z) = \pi/3$.

### Part e
Expand the expression:
$$\frac{x^2 - 2x + 1 + y^2 + 4y + 4}{2} \geq x + y$$
$$x^2 - 2x + 1 + y^2 + 4y + 4 \geq 2x + 2y$$
$$x^2 - 4x + y^2 + 2y + 5 \geq 0$$
Complete the square for $x$ and $y$:
$$(x^2 - 4x + 4) + (y^2 + 2y + 1) \geq 0$$
$$(x - 2)^2 + (y + 1)^2 \geq 0$$
Since squares of real numbers are always non-negative, the sum of two squares is $\geq 0$.
**Conclusion:** The inequality holds for all $x, y \in \mathbb{R}$.
