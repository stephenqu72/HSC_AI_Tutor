To evaluate the integral $\int \frac{\sin^3 x - \cos^3 x}{\sin x - \cos x} dx$, we proceed using algebraic simplification and trigonometric identities.

### Step 1: Simplify the integrand using algebraic factorization
The numerator is a difference of two cubes, which follows the identity $a^3 - b^3 = (a - b)(a^2 + ab + b^2)$.
Setting $a = \sin x$ and $b = \cos x$:
$$\sin^3 x - \cos^3 x = (\sin x - \cos x)(\sin^2 x + \sin x \cos x + \cos^2 x)$$

Substitute this back into the integrand:
$$\frac{(\sin x - \cos x)(\sin^2 x + \sin x \cos x + \cos^2 x)}{\sin x - \cos x} = \sin^2 x + \sin x \cos x + \cos^2 x$$
(Assuming $\sin x - \cos x \neq 0$).

### Step 2: Use trigonometric identities
We know the Pythagorean identity: $\sin^2 x + \cos^2 x = 1$.
The expression simplifies to:
$$1 + \sin x \cos x$$

To make this easier to integrate, we use the double-angle formula $\sin 2x = 2 \sin x \cos x$, which implies $\sin x \cos x = \frac{1}{2} \sin 2x$:
$$1 + \frac{1}{2} \sin 2x$$

### Step 3: Integrate
Now, integrate the simplified expression with respect to $x$:
$$\int (1 + \frac{1}{2} \sin 2x) dx = \int 1 dx + \frac{1}{2} \int \sin 2x dx$$
$$= x + \frac{1}{2} \left( -\frac{1}{2} \cos 2x \right) + c$$
$$= x - \frac{1}{4} \cos 2x + c$$

If we look at option A: $x - \frac{1}{2}\sin^2 x + c = x - \frac{1}{2}(\frac{1 - \cos 2x}{2}) + c = x - \frac{1}{4} + \frac{1}{4}\cos 2x + c$. This does not match.

$\int (\sin^2 x + \sin x \cos x + \cos^2 x) dx = \int (1 + \sin x \cos x) dx = x + \frac{1}{2} \sin^2 x + c$ (since $\frac{d}{dx}(\frac{1}{2}\sin^2 x) = \frac{1}{2} \cdot 2\sin x \cos x = \sin x \cos x$).
